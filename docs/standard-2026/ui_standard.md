# DNA Visual: Premium 2026 v4.3 (Navy Mastery)

Este documento define la base atómica del diseño institucional de **Gestión Tómbola**.

## 🎨 Paleta de Colores Corporativa

### Primario: Navy Institucional
- **Hex**: `#1B3B5A`
- **Uso**: Títulos, botones primarios, iconos de cabecera, estados activos (Sidebar).
- **Semántica**: Autoridad, confianza y solidez.

### Neutros y Superficies
- **Superficie Base**: `bg-slate-50` (Gris etéreo para el fondo general).
- **Cartas/Paneles**: `bg-white` (Blanco puro con sombras suaves).
- **Bordes**: `border-slate-100` o `border-slate-200/50`.
- **Textos**: 
  - `text-primary` (Navy `#1B3B5A`) para títulos.
  - `text-slate-600` para cuerpo de texto.
  - `text-slate-400` para metadatos y placeholders.

---

## 🆎 Tipografía y Pesos

El sistema utiliza una escala dual para diferenciar niveles de información:

### 1. Títulos y Marca (`Manrope`)
- **Uso**: `h1`, `h2`, `h3`, `h4`.
- **Estilo**: `font-black` (900) con tracking ajustado.
- **Color**: Siempre en `text-primary`.

### 2. Interfaz y Datos (`Inter` / `Sans`)
- **Uso**: Filas de tabla, inputs, etiquetas, párrafos.
- **Estilo**: 
  - `font-bold` para datos de usuario.
  - `font-medium` para labels (11px, Uppercase, Tracking-widest).

---

## 📐 Geometría y Sombras

### Redondeo Institucional (`rounded-premium-card`)
- **Cartas**: `rounded-[48px]` para una estética suave y premium.
- **Inputs/Botones**: `rounded-2xl` a `rounded-premium-input`.

### Sombras (`shadow-premium`)
- **Base**: `shadow-[0_8px_30px_rgb(0,0,0,0.015)]`.
- **Acción**: `shadow-xl shadow-primary/5` en hover.
